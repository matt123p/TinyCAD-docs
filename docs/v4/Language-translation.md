# Language Translation

TinyCAD can be switched between several interface languages.

## Available languages

The current language options are:

| Code | Language |
|---|---|
| `en` | English |
| `fr` | Français |
| `es` | Español |
| `de` | Deutsch |
| `it` | Italiano |
| `ja` | 日本語 |
| `zh-CN` | 中文 (简体) |

## Translation quality and feedback

These translations are maintained by AI.

If you notice awkward wording, incorrect technical terms, or untranslated text,
user feedback is very welcome. Suggestions for additional translation languages
are also welcome.
