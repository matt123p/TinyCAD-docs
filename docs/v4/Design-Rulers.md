# Design Rulers

Design Rulers provide positional reference around the drawing so placement and checking are easier.

## What Design Rulers are for

- Reading approximate position while placing symbols, wires, and annotations.
- Cross-checking location together with **Show Position** when you need exact
	coordinates.
- Keeping layout consistent across edits and reviews.

## Units and coordinates

Ruler units follow the global units setting:

- **Units in inches** or **Units in mm** (from grid/settings options).
- The same unit choice is reflected by both the rulers and the **Show
	Position** dialog.

## Related workflows

- Use **Show Grid** and **Snap to Grid** with rulers for cleaner placement and
	reliable wire/pin alignment.
- When exporting the whole design as a bitmap (with no selection), TinyCAD
	includes both design details and design rulers in the output.

## Design Guides

Design guides are visual alignment references you can use while laying out a
schematic.

- Use them to keep repeated sections aligned consistently across the page.
- Combine guides with rulers for quick visual spacing checks.
- Keep guides aligned to your chosen grid spacing to avoid off-grid placement.
- Use **Show Position** when guide-based visual placement needs exact
	coordinate confirmation.

## Practical tips

- Keep a consistent unit setting for a project to avoid placement confusion.
- If you need exact placement checks, open **Show Position** while editing.
- Re-check page setup before print/export so ruler-based spacing matches your
	intended output.
