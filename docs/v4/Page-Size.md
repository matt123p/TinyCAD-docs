# Page Size

Use **Page Size** to set the sheet dimensions for all sheets.

<img src="/v4/images/media/image4.png" alt="The page size dialog" />

## What you can change

- **Paper Size**: choose a standard paper size such as A4.
- **Orientation**: select **Portrait** for a taller page or **Landscape** for a wider page.
- **Width** and **Height**: enter a custom page size in millimeters.

When you choose a standard paper size, TinyCAD fills in the width and height for
you. Changing the orientation swaps those dimensions.

## Typical use

Use page size to match the kind of output you want:

- choose a standard size for printing or PDF export
- use landscape for wider schematics
- use a custom width and height when a standard paper size does not fit your drawing

The selected page size is saved with the sheet as part of the drawing.
