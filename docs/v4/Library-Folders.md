# Library Folders

## Overview

The My Libraries panel supports folders so you can organise installed libraries into groups.


<div class="manual-tabset" data-tabset="product-mode" data-tabset-label="Choose your TinyCAD product" data-default-tab="Browser" markdown="1">

### Browser

Library folders are not supported in the Browser version of TinyCAD.

### Desktop App

Folders are saved together with your library list, so the structure is preserved the next time you open TinyCAD.

Library folders are only available in the desktop app.

Browser-based modes still show My Libraries where supported, but they do not expose folder creation, folder grouping, or folder drag-and-drop.

#### What You Can Do

- Create folders to group related libraries
- Rename folders
- Delete folders
- Reorder folders by drag and drop
- Move libraries between folders by drag and drop
- Add multiple existing library files in one operation
- Add libraries directly into the currently selected folder

Libraries that are not assigned to a folder appear in the Unfiled section.

#### Create a Folder

1. Open the Libraries panel.
2. Switch to My Libraries.
3. Select the folder button in the panel header.
4. Enter a name for the new folder.

The new folder appears in the library list and can be selected immediately.

#### Select a Folder

Click a folder row to select it.

The selected folder is used when you add existing libraries.

If you click a library, TinyCAD automatically selects the folder that library belongs to. This makes it clear where newly added libraries will be placed.

#### Add Libraries to a Folder

1. Select the target folder in My Libraries.
2. Select Add.
3. Choose one or more existing library files.

The selected libraries are added into the selected folder.

If you do not select a custom folder, libraries are added to Unfiled.

#### Move Libraries Between Folders

Drag a library from its current location and drop it onto another folder.

You can also drop a library onto Unfiled to remove it from a custom folder.

If a folder is empty, TinyCAD shows a drop target so you can move libraries into it.

#### Reorder Folders

Folders can be reordered directly in the panel.

1. Hover over the folder you want to move.
2. Drag the reorder handle.
3. Drop the folder onto the target folder position.

The new order is saved automatically.

#### Rename or Delete a Folder

The folder action buttons are shown when a folder is hovered or selected.

- Select Edit to rename the folder.
- Select Delete to remove the folder.

When a folder is deleted, any libraries inside it are moved to Unfiled. The library files themselves are not removed from TinyCAD.

#### Unfiled Libraries

Unfiled is the default location for libraries that are not assigned to a custom folder.

- New libraries go to Unfiled if no custom folder is selected
- Libraries from a deleted folder are moved to Unfiled
- You can drag libraries from Unfiled into any custom folder

When no custom folders exist, TinyCAD simply shows the library list without an Unfiled heading.

#### Notes

- Folder organisation changes are stored with your saved library configuration
- Adding multiple libraries at once is supported when choosing existing library files
- Folder order and library-to-folder assignments persist between sessions
