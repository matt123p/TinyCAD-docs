# Installation Instructions

This page covers the installation of TinyCAD.

## Install on Windows

For Windows, the preferred way to install TinyCAD v4 is from the
[Microsoft Store](https://apps.microsoft.com/detail/9NZQ1M56NM3V).

If you are using Windows, use the Microsoft Store version unless you have a
specific reason to install from another source.

The [GitHub releases](https://github.com/matt123p/TinyCADv4/releases) also
includes a Windows installer.

## Install on macOS

For macOS, download TinyCAD v4 from the
[GitHub releases](https://github.com/matt123p/TinyCADv4/releases).

## Install on Linux

TinyCAD is available through the [Snap Store](https://snapcraft.io/tinycad) for 
mainstream snap-enabled Linux distributions including Ubuntu, Kubuntu, KDE Neon, Pop!_OS, elementary OS,
Debian, Fedora, openSUSE, Manjaro, Arch Linux, CentOS, and Red Hat Enterprise
Linux.

On Ubuntu and many Ubuntu-based distributions, snap support is already present.
In that case you can install TinyCAD directly with:

```bash
sudo snap install tinycad
```
If this doesn't work then the TinyCAD page on the [Snap Store](https://snapcraft.io/tinycad) has 
links to distribution-specific setup instructions for each supported Linux distribution.

The [GitHub releases](https://github.com/matt123p/TinyCADv4/releases) also
includes Linux `.deb` packages.

## Use on ChromeOS and Chromebooks

On ChromeOS use [TinyCAD Web](../TinyCAD-Web-overview-and-quick-start/)
instead of installing the desktop application.
