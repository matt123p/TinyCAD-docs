# TinyCAD Web overview and quick start

## TinyCAD for the Web

**Note:** Historically this was published as a pre-release and supplied "as-is".

TinyCAD for the web runs directly in your browser, does not need installing,
and works on operating systems that support a compatible browser. The file
formats are compatible with desktop TinyCAD (`.dsn` drawings and `.tclib`
libraries).

## Before you start

Make sure you have the following:

- Chrome or Edge (for local-file mode)
- For Google Drive mode, Chrome is recommended; Firefox can also work
- Read the Terms and Conditions: [Terms and Conditions](/TCs)

It is intended to work with Windows, Mac, Chromebooks, and Linux.

## Entry points

- Local file mode: <https://online.tinycad.net/fs>
- Google Drive mode: <https://online.tinycad.net/ga>
- Support: <https://discord.gg/bdXnjhrSYQ>

## Getting started (saving drawings on local computer)

This mode relies on browser features available in modern Chrome or Edge.
Use the latest version of either browser, then open:

- <https://online.tinycad.net/fs>

## Getting started (saving drawings on Google Drive)

Before you start, you must have a Google account.

First-time setup:

1. Go to <https://online.tinycad.net/ga>.
2. If you have an ad blocker, turn it off for this site (it can interfere with
   Google's sign-in process).
3. If no popup appears, check whether popup blocking is enabled and allow
   popups for the site.
4. Sign in to your Google account in the popup (if required).
5. Tick **all** consent checkboxes to allow full Google Drive integration.
6. If Google sign-in flow fails after accepting terms, reload the page and try
   again.

## Privacy

### Local file mode

TinyCAD runs entirely in your browser and stores work locally in the browser.
No part of your work or other files is sent to TinyCAD servers.

### Google Drive mode

Drawings are stored in your Google Drive. On first use, you must grant access
for TinyCAD to your Google files to enable integration.

TinyCAD still runs in-browser and writes data to your private Google Drive; no
part of your work or other files is sent to TinyCAD servers.

TinyCAD uses Google Analytics to track site usage volumes.
